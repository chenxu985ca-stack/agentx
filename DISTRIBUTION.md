# 获客与分发策略

## 现状：零粉丝起步

没有粉丝不是劣势——AI Agent 赛道现在是最佳入场时机。核心策略：**用内容证明能力，用结果吸引客户。**

---

## 渠道矩阵（按优先级排序）

### 🔥 即刻 (Jike) — 第一站
- **优势：** AI/开发者密度最高的中文社区，容忍度好，冷启动友好
- **动作：**
  - 每天发一条「AI Agent 实战日记」，记录你用 Claude Code + Hermes 做了什么
  - 格式：截图 + 三句话做了什么 + 一句心得
  - 标签：#AI Agent #ClaudeCode #独立开发
  - 示例："用 Hermes 搭了一个自动查竞品的 cron 任务，每天早上 9 点微信推送。全程没写一行代码。这就是 AI Agent 的力量。"
- **目标：** 前 100 个粉丝，3-5 个咨询

### 📕 小红书 — 流量蓝海
- **优势：** AI 教程类内容紧缺，流量推荐机制对新号友好
- **内容方向：**
  - 「手把手教你装 Hermes Agent」系列（图文）
  - 「Claude Code 让我一天写完一个网站」
  - 「两个 AI 助手协同工作是什么体验」
  - 封面风格：干净、技术感、大字突出
- **频率：** 每周 2-3 篇
- **引流方式：** 个人简介放联系方式，评论区引导私信

### 📝 知乎 — 信任背书
- **优势：** 搜索流量长尾效应强
- **动作：**
  - 回答「AI Agent 有什么用？」「Hermes Agent 怎么样？」类问题
  - 每篇回答末尾放服务介绍 + 联系方式
  - 写 3-5 篇深度专栏文章作为内容资产

### 💬 社群渠道
- **微信群：** 加入 AI/独立开发者社群，先贡献价值再自然转化
- **V2EX / 电鸭 / w2solo：** 独立开发者聚集地，发帖分享 AI Agent 搭建经验
- **GitHub：** 给 Hermes/Claude Code 项目提 PR 或写教程，建立权威

### 🌍 海外渠道（后期）
- **Product Hunt：** 把服务包装成「AI Agent Setup Service」上线
- **Twitter/X：** 英文内容同步，分享 AI Agent setup 经验

## 内容的「钩子」

每天用 Claude Code + Hermes 做的事情就是最好的内容素材：

| 你的日常 | 转化内容 |
|----------|----------|
| 用 Claude Code 写了个网站 | 「AI 帮我写了一个完整网站，花了 2 小时」 |
| Hermes 定时搜竞品 | 「我让 AI 每天早上给我发竞品日报」 |
| 配好了双 Agent 协同 | 「两个 AI 聊天，我在旁边看呆了」 |
| 帮朋友装好了 Hermes | 「朋友说这是他花过最值的 500 块」 |


## 定价心理学

- **基础版 ¥599：** 锚定价格，让标准版看起来超值
- **标准版 ¥1,999：** 主力利润款，约 70% 客户选这个
- **企业版 ¥3,999：** 提升品牌天花板，偶尔出一单就很赚

## 长期护城河

1. **案例积累：** 每个客户做完写一个 case study（脱敏）
2. **技能库沉淀：** Hermes 越用越聪明，你的技能库就是你的护城河
3. **口碑复利：** 做好的客户会推荐给朋友，这是最高质量的获客
4. **产品化：** 后期把服务打包成「一键部署脚本」卖工具而非卖时间

# AgentX 快速使用指南

> 基础版交付文档 · 适用 Hermes 单 Agent 用户

---

## 1. 启动 Hermes

打开终端（Terminal），输入：

```bash
hermes
```

看到类似下面的界面就说明启动成功了：

```
Hermes v0.15.1 — Ready
Type /help for commands
>
```

---

## 2. 常用命令

| 命令 | 作用 |
|------|------|
| `/help` | 查看所有命令 |
| `/model` | 切换模型 |
| `/clear` | 清空当前对话 |
| `/exit` 或 `Ctrl+C` | 退出 Hermes |
| `/skills` | 查看已安装的技能列表 |

---

## 3. 日常使用

### 直接提问

像聊天一样直接输入问题，Hermes 会回复你：

```
> 帮我总结一下今天的热点新闻
> 这段代码是什么意思？（直接粘贴代码）
> 帮我把这段话翻译成英文
```

### 切换模型

如果回复速度慢或想用其他模型：

```
/model
```

然后用方向键选择模型，回车确认。

- **deepseek-v4-pro**（默认）— 综合能力强，性价比高
- **deepseek-chat** — 更快，适合简单任务

### 中断回复

如果回复太长或方向不对，按 `Ctrl+C` 可以中断当前回复。

---

## 4. 进阶技巧

### 长文本处理

Hermes 可以一次性处理很长的文本。直接粘贴文章、代码、聊天记录都行，不需要分段。

### 多轮对话

Hermes 会记住当前对话的上下文。如果话题换了，建议先用 `/clear` 清掉旧上下文再问，避免混淆。

### 追问

回复不满意？直接追问：
- "再详细一点"
- "换个角度解释"
- "给我举个例子"

---

## 5. 常见问题

### Q: 启动报错 "API key not found"

检查 `~/.hermes/.env` 文件里有没有这一行：

```
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxx
```

### Q: 回复很慢

1. 检查网络
2. 试试 `/model` 切换到 `deepseek-chat`（更快）
3. 不要在高峰期同时跑大量任务

### Q: 回复内容不对

1. `/clear` 清空上下文后重试
2. 把问题描述得更具体一些
3. 追问纠正："不对，我要的是XXX"

### Q: 怎么更新 Hermes

```bash
hermes update
```
